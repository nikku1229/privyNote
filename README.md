# PrivyNote

PrivyNote is a secure personal diary application that allows users to write, edit, delete, and view diary entries. Each user has a private space and can access it from any device.

## Features

- User registration and login
- Private diary per user
- Create, edit, delete, and view notes
- Auto-save while typing
- Search diary entries
- Secure JWT-based authentication
- Responsive UI for desktop and mobile

## Tech Stack

- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- Deployment: GitHub (and later Vercel)

## Setup

1. Clone the repo:

```bash
git clone https://github.com/<your-username>/PrivyNote.git

cd server
npm install

cd ../client
npm install

```
